* Local Repo Work Flow *
1. create new repo in Github
2. clone new repo in local repo using git clone "url"
3. changes :- Add new file or create new file or folder into repo
4. Add changes history using :- git add . / all changes / git add fileName
5. commit :- commit change 
6. push :- push into remote repository

*upload project in Github*
1. create new repo in Github
2. git remote add origin link
3. git add .
4. git commit -m "initial commits"
3. git push origin main
